# istina

**istina** (Slavic *истина* — truth) is a Rust validation library: composable [`Validate`](https://docs.rs/istina) traits,
proof types `Validated<T>`, structured errors with RFC 6901 field paths, and a serializable `Rule` engine for JSON and config.

> **Not affiliated** with [MSU Istina](https://istina.msu.ru) or [istina-dev](https://github.com/istina-dev).
